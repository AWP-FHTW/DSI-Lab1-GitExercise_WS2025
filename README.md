# Hello Rust

A simple "Hello, World!" example in Rust, based on the [Rust Getting Started guide](https://rust-lang.org/learn/get-started).

## Running the project

To run this project, you need to have [Rust](https://www.rust-lang.org/tools/install) installed.

Then simply run:

```bash
cargo run
```

This will compile and run the program, printing "Hello, world!" to the console.
